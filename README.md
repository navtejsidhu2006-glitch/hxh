# Hunter x Hunter: Trails of the Exam (Text RPG)

A text-based, stylized Hunter x Hunter RPG inspired by the event-driven flow of **Life in Adventure**.

## Features

- Character setup with origin + aura tendency.
- D20-style checks (`STR`, `AGI`, `INT`, `WILL`, `CHA`) and branching outcomes.
- Event loop with moral choices, fame, zenny rewards, and flags that influence ending.
- Lightweight turn-based combat (basic attacks, nen burst, defend, item usage).
- Multiple endings based on your reputation and decisions.

## Run

```bash
python3 game.py
```

## Notes

- Built as a terminal-first experience.
- Uses only Python standard library.
